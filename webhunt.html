This is rgb values

A hex code is a 6 diget code that represents a color

A color name is the name of a color

Hue is a degree on the color wheel

Saturation is a percentage value, 0% represents grey and 100% represents full color

Lightness is also a percentage, 0% is full black and 100% represents white

Contrast is the difference and connection in lightness

Opacity is the measure of how translucent a thing is

HSLA stands for hue, saturation, lightness, alpha. Alpha is the opacity value, 0 is fully transparent, 1 is fully opaque
